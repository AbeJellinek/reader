EPUB view demo (doesn't work on Safari)

  • This view is just loading a demo HTML file in an iframe.
  • The whole UI including popups is implemented outside the view. That includes:
    • Sidebar.
    • Toolbar.
    • Context menus.
    • Annotation popup (only visible when sidebar is closed).
    • Text selection popup (5 colors popup to highlight text).
    • Overlay popup (displays extra information when hovering an overlay, i.e. link).
    • Find popup.
  • Highlight and note annotations can be created but would only appear in the sidebar, because this demo view doesn't implement annotation renderer.
  • There are two hard-coded annotations and they can be clicked to select an annotation in the sidebar or open a popup (if sidebar is closed).
  • Split view can be enabled over context menu.
  • Keyboard shortcuts:
    • First tab press focuses annotation/link and allows to navigate between them with arrow keys.
    • Second tab escapes from the view and focuses toolbar or the second view.
    • Zoom In (Cmd-Minus) / Zoom Out (Cmd-Plus) / Zoom Reset (Cmd-0).
  • Annotation or text selection dragging/copying works as in PDF reader (supports Quick Copy or dropping into note-editor).
  • Supports outline view in the sidebar.
  • External link zotero.org (try to hover it to see a popup).
